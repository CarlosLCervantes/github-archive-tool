This is a command line tool for downloading files from th github archive. You can specify the date range that you want and it will pull down all the files for the date. It also shows the most popular repositories for the time frame.

Example Usage
==========================
gh_repo_stats.rb --after 2012-11-01T13:00:00Z --before 2012-11-02T03:12:14-03:00 --event PushEvent --count 42
